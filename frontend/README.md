This is the StratIQ frontend, a [Next.js](https://nextjs.org) app.

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — it redirects to `/dashboard` by default.

The frontend talks to the FastAPI backend at the URL set in `NEXT_PUBLIC_API_URL` (defaults to `http://localhost:8000/api/v1`).
